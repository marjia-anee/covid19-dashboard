# Covid19-Dashboard


## Overview

This tracker aims to show the numbers & trends of people affected, recovered and deceased due to COVID-19 - Data classified under 3 categories.

Tamil Nadu (All Districts)
India (All States/UTs)
World (All Countries)

### Technologies

- React.js
- Chart.js
- Material UI 
- Axios 
- Netlify
